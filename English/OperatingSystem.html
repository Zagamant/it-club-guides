Operating System

An operating system, or OS, is a software program that enables the computer hardware to communicate and operate with the computer software. Without a computer operating system, a computer would be useless.

In computing OS linked between hardware and user, which is responsible for the management and coordination of activities and sharing resources of a computer, that acts as a host for computing applications run on the machine. One of the purposes of an OS is to handle the resource allocation and access protection of the hardware. This relieves the programmers from having to manage these details.

At the simplest level, an operating system does two things. It manages the hardware and software resources of the system. In a desktop computer, these resources include such things as the CPU, memory, disk space and more. It provides a stable, consistent way for applications to deal with the hardware without having to know all the details of the hardware. Within the broad family of operating systems, there are generally four types, categorized based on the types of computers they control and the sort of applications they support.

Real-time operating system (RTOS) – Real-time operating systems are used to control machinery, scientific instruments and industrial systems. An RTOS typically has very little user-interface capability, and no end-user utilities, since the system will be a “sealed box” when delivered for use. A very important part of an RTOS is managing the resources of the computer so that a particular operation executes in precisely the same amount of time, every time it occurs. In a complex machine, having a part move more quickly just because system resources are available may be just as catastrophic as having it not move at all because the system is busy.

Single-user, single task – As the name implies, this operating system is designed to manage the computer so that one user can effectively do one thing at a time.

Single-user, multi-tasking – This is the type of operating system most people use on their desktop and laptop computers today. Microsoft’s Windows and Apple’s MacOS platforms are both examples of operating systems that will let a single user have several programs in operation at the same time. For example, it’s entirely possible for a Windows user to be writing a note in a word processor while downloading a file from the Internet while printing the text of an e-mail message.

Multi-user – A multi-user OS allows many different users to take advantage of the computer’s resources simultaneously. The OS must make sure that the requirements of the various users are balanced, and that each of the programs they are using has sufficient and separate resources so that a problem with one user doesn’t affect the entire community of users. Unix, VMS and mainframe operating systems, such as MVS, are examples of multi-user operating systems.

Examples of most popular operating systems : Windows XP is an operating system produced by Microsoft for use on personal computers, including home and business desktops, laptops, and media centers. It was released in 2001. Windows XP is the successor to both Windows 2000 Professional and Windows Me, and is the first consumer-oriented operating system produced by Microsoft to be built on the Windows NT kernel and architecture.

MS-DOS is an operating system for x86 based personal computers, which was purchased by Microsoft. It was the most commonly used member of the DOS family of operating systems, and was the main operating system for personal computers during the 1980s and 1990s. MS-DOS developed out of QDOS (Quick and Dirty Operating System), also known as 86-DOS . Several versions were released under different names for different hardware. MS-DOS had eight major versions released before Microsoft stopped development in 2000.

Операционная система

Операционная система, или ОС, представляет собой ПО, которое позволяет компьютерному оборудованию взаимодействовать и работать с компьютерным ПО. Без компьютерной ОС компьютер был бы бесполезен.

Во время работы ОС связана между аппаратным обеспечением и пользователем, который отвечает за управление и координацию деятельности и совместное использование ресурсов компьютера, который выступает в качестве хоста для вычислительных приложений, запущенных на машине. Одной из целей OC является управление распределением ресурсов и защитой доступа к оборудованию. Это избавляет программистов от необходимости управлять этими деталями.

На самом простом уровне, операционная система делает две вещи. Он управляет аппаратными и программными ресурсами системы. В настольном компьютере эти ресурсы включают такие вещи, как процессор, память, дисковое пространство и многое другое. Это обеспечивает стабильный, последовательный способ для приложений, чтобы иметь дело с оборудованием без необходимости знать все детали оборудования. В широком семействе операционных систем, как правило, существует четыре типа, классифицированных на основе типов компьютеров, которыми они управляют и вид приложений, которые они поддерживают.

Операционная система реального времени (RTOS)-операционные системы реального времени используются для управления машинами, научными приборами и промышленными системами.RTOS обычно имеет очень мало возможностей пользовательского интерфейса и никаких утилит для конечных пользователей, поскольку система будет “запечатанной коробкой” при поставке для использования. Очень важной частью RTOS является управление ресурсами компьютера таким образом, что определенная операция выполняется каждый раз одно и то же количество времени. В сложном компьютере, если одна часть работает быстрее просто потому, что системные ресурсы это ей позволяют, это может быть столь же катастрофическим, как и отсутствие работы вообще, потому что система перегружается.

Однопользовательская-как следует из названия, эта операционная система предназначена для управления компьютером, чтобы один пользователь мог эффективно делать одну вещь за один раз.

Однопользовательская, многозадачность-это тип ОС, которую большинство людей используют на своих настольных и портативных компьютерах сегодня. Платформы Microsoft Windows и Apple MacOS являются примерами операционных систем, которые позволят одному пользователю иметь несколько программ в работе одновременно. Например, пользователь Windows вполне может писать заметку в текстовом редакторе во время загрузки файла из интернета при печати текста сообщения электронной почты.

Многопользовательская-многопользовательская ОС позволяет многим пользователям одновременно пользоваться ресурсами компьютера. OC должна убедиться, что требования различных пользователей сбалансированы, и что каждая из программ, которые они используют, имеет достаточные и отдельные ресурсы, так что проблема с одним пользователем не влияет на все сообщество пользователей. Unix, виртуальные машины и мэйнфреймы операционных систем, таких как MVS, являются примерами многопользовательских операционных систем.

Примеры наиболее популярных операционных систем : Windows XP-это операционная система, выпускаемая корпорацией Майкрософт для использования на персональных компьютерах, включая домашние и деловые настольные компьютеры, ноутбуки и медиацентры. Он был выпущен в 2001 году. Windows XP является преемником Windows 2000 Professional и Windows Me, и является первой ориентированной на потребителя операционной системой, произведенной Microsoft, которая будет построена на ядре и архитектуре Windows NT. MS-DOS-это операционная система для персональных компьютеров на базе x86, приобретенная компанией Microsoft. Это был наиболее часто используемый член семейства операционных систем DOS, и был основной операционной системой для персональных компьютеров в течение 1980-х и 1990-х гг. MS-DOS развивалась из QDOS (быстрая и грязная операционная система), также известный как 86-DOS. Несколько версий были выпущены под разными названиями для разных аппаратных средств. MS-DOS имела восемь основных версий, выпущенных до того, как Microsoft прекратила разработку в 2000 году.
