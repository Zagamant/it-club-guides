Функция scanf и ее применение для ввода данных

Функция scanf() предназначена для ввода данных в программу. Она может считывать данные всех базовых типов и автоматически конвертировать их в нужный внутренний формат. Данная функция, как и функция printf(), находится в библиотеке <stdio.h>

Так же как для функции printf(), для функции scanf() в кавычках указывается модификаторы переменных, а после запятой необходимо указать список переменных. Основное различие двух этих функций заключается в особенности данного списка. Функция printf() использует имена переменных, в то время как функция scanf() - только указатели на переменные. К счастью, при применении этой функции мы можем ничего не знать об указателях. (Про указатели вы узнаете немного позже). Необходимо помнить, что если вам нужно ввести некоторое значение и присвоить его переменной одного из основных типов, то перед именем nepеменной требуется писать символ &.

Рассмотрим несколько простых примеров и все станет понятно:

int a; //объявляем переменную a
printf("enter a: "); //данную строчку рекомендую написать, чтобы было понятно в какой момент необходимо вводить данные.
scanf("%d",&a); //вводим зничение в перенную a, ввод числа заканчивается клавишей enter или пробелом.
double b,c; //объявляем две переменные вещественного типа.
printf("enter b, c: "); //сообщение для пользователя, что ожидается ввод двух чисел.
scanf("%lf%lf", &b, &c); //при вводе сразу двух переменных модификаторы переменных указываются без пробела. Никаких символов, кроме модификаторов в кавычках быть не должно.

Арифметические операторы

Операция Символ Название
сложение + плюс
вычитание - дефис
умножение * звездочка
деление / слэш
остаток от целочисленного деления % процент
инкремент(увеличение переменной на 1) ++ два знака + без пробела
декремент(уменьшение на 1) -- два знака - без пробела

Зная арифметические операции, функции ввода и вывода, можно приступить к решению простейших задач

Задачи

  • Программа определяет ваш возраст. Требует ввести год вашего рождения и нынешний год.

    • 						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main()
{
	int year, yearOfBirth, age; //объявлены переменные нынешний год, год рождения и возраст
	printf("Enter this year: ");
	scanf("%d", &year);//ввод нынешнего года
	printf("Enter year of birth: ");
	scanf("%d", &yearOfBirth);//ввод года рождения
	age = year - yearOfBirth; //вычисляется возраст
	printf("Age: %d\n", age); //вывод возраста
	return 0;
}
  • Дан диаметр окружности d. Найти ее длину l=PI*d. В качестве значения PI использовать 3.14.

    • 						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main()
{
	const double PI = 3.14; //константное число PI
	double d, l; //диаметр и длина окружности
	printf("Enter diameter: ");
	scanf("%lf", &d);//ввод диаметра 
	l = d * PI; //вычисляется длина окружности
	printf("Length: %lf\n", l); //вывод длины
	return 0;
}
  • Даны стороны прямоугольника. найти периметр и площадь

    • 						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main()
{
	int a, b; //стороны прямоугольника
	printf("a = ");
	scanf("%d", &a); //ввод a
	printf("b = ");
	scanf("%d", &b); //ввод b
	printf("P = %d\n", (a+b)*2); //вычисление и вывод периметра
	printf("S = %d\n", a*b);     //вычисление и вывод площади
	return 0;
}
