Актуальность проблемы генерации случайных и псевдослучайных последовательностей.

Генерация псевдослучайных чисел – это сложная математическая задача. Задача генерации случайных чисел на классическом процессоре не может быть решена, так как работа компьютера детерминирована по определению. Тем не менее, можно сгенерировать очень длинные наборы чисел такие, что будут иметь те же свойства, что и наборы истинно случайных чисел. Случайными числами мы будем называть псевдослучайные числа.

Для генерации случайных чисел используется функция rand. Данная функция находится в библиотеке stdlib.h.

						
#include <stdio.h>
#include <stdlib.h> //библиотека для генерации случайных чисел
 
int main()
{
	int r;
	for (int i = 0; i < 10; i++)
	{
		r = rand();  //генерация случайного числа       
		printf("%d\n", r);
	}
	return 0;
}

Если несколько раз запустить данную программу, то вы заметите, что все время генерируются одни и те же числа.

Чтобы были различные числа, необходимо инициализировать генератор случайных чисел (ГСЧ, или RNG - random number generator), задать зерно – seed, на основе которого в дальнейшем будет происходить генерация. Важно, что для одного и того же начального значения генератор будет возвращать одни и те же числа. Инициализируется генератор случайных чисел функцией srand.

						
srand(42);

Присваиваем переменной r случайное число 

						
r = rand();

Значение будет лежать в диапазоне от 0 до RAND_MAX.
Если необходимо сгенерировать число в промежутке от a до b 

						
r = rand()%(b-a+1) +a;

Для того, чтобы при следующем запуске получить новый набор чисел, нужно инициализировать генератор каждый раз разными значениями. Например, можно использовать системное время - функция time. Данная функция находится в библиотеке time.h.

						
srand(time(NULL)); //используется текущее время в милисекундах

Заполнение массивов случайными числами.

Следующая программа заполняет целочисленный массив случайными числами и выводит его:

						
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
int main()
{
	const int n=10;
	int mas[n];
	srand(time(NULL));
	for (int i = 0; i < n; i++) 
	{
		mas[i]= rand()%100;
		printf("%d\t", mas[i]);
	}
	return 0;
}

Задачи

  • Создать массив, заполнить случайными числами, удалить в массиве все числа, которые повторяются более двух раз. 

    • 						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
int main()
{
	int a[10];
	int n = 10;
	int count = 0;
	srand(time(NULL));
	for (int i = 0; i < n; i++)
	{
		a[i] = rand() % 20;
		printf("%d\t", a[i]);
	}
	printf("\n");

	for (int i = 0; i < n; i++)
	{
		count = 1;

		for (int j = i + 1; j < n; j++)
		{
			if (a[j] == a[i])
			{
				count++;
			}
		}

		if (count > 2)
		{
			int one = a[i];
			for (int j = i; j < n; j++)
			{
				if (a[j] == one)
				{
					// удаляем
					for (int k = j + 1; k < n; k++)
					{
						a[k - 1] = a[k];
					}
					n--;
					j--;
				}
			}
			i--;
		}
	}
	if (n == 0)
	{
		printf("empty array\n");
	}
	else
	{
		for (int p = 0; p < n; p++)
		{
			printf("%d\t",a[p]);
		}
	}
	printf("\n");

	
	return 0;
}
