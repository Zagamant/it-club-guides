Рекурсивные функции и их применение.

Рекурсивная функция - это функция, которая вызывает сама себя.

Простым примером применения рекурсии является функция factorial(), вычисляющая факториал целого числа. Факториалом числа n является произведение чисел от 1 до n. Например, факториал 3 равен 1*2*3 или 6.

						
//рекурсивная функция
int factorial(int n) 
{
	int answer;
	if(n==1)
	{ 
		return 1;  //в случае если n=1
	}
	answer = factorial(n-1)*n; //рекурсивный вызов функции с параметром n-1
	return answer;
}

//нерекурсивная функция
int fact (int n)   
{
	int answer = 1;
	for(int t=1; t<=n; t++)
	{
		answer = answer * t;
	}
	return answer;
}

Работа нерекурсивной функции кажется очевидной. Циклом перемножаются все числа от 1 до n.

Действие рекурсивной функции factorial() немного сложнее. Когда factrorial() вызывается с аргументом 1, функция возвращает 1, во всех остальных случаях она возвращает произведение factorial(n- 1) * n. Для вычисления этого значения factorial() вызывается с параметром n-1. Это происходит, пока n не станет равно 1.

При вычислении факториала числа 2, первый вызов factorial() приводит ко второму вызову с аргументом 1. Данный вызов возвращает 1, после чего результат умножается на 2 (исходное значение n). Ответ, таким образом, будет 2. Можно попробовать вставить printf() в factorial() для демонстрации уровней и промежуточных ответов каждого вызова.

При работе с рекурсией нужно быть осторожным и предусмотреть условие выхода из функции, иначе получим бесконечную рекурсию, ситуация так же опасна как и бесконечный цикл.

Когда функция вызывает сама себя, в стеке выделяется место для новых локальных переменных. Код функции работает с данными переменными. Рекурсивный вызов не создает новую копию функции, новыми являются только аргументы.

Рекурсивные версии большинства подпрограмм могут выполняться немного медленнее, чем их варианты с циклом, поскольку к необходимым действиям добавляются вызовы функций. Но в большинстве случаев это не имеет значения. Много рекурсивных вызовов в функции может привести к переполнению стека. Поскольку местом для хранения параметров и локальных переменных функции является стек и каждый новый вызов создает новую копию переменных, пространство стека может исчерпаться. Если это произойдет, то возникнет ошибка - переполнение стека.
Основным преимуществом применения рекурсивных функций является написание более простого кода.

Если вы не до конца понимаете принцип работы рекурсивной функции лучше воспользуйтесь обычным циклом.

Введение в указатели.

Указатель - это переменная, которая хранит адрес другой переменной.

В таблице приведено сравнение обычной переменной и указателя.

Название Объявление Инициализация Значение Адрес Вывод значения Вывод адреса
переменная int x; x = 10; x &x printf("%d", x); printf("%d", &x);
указатель int* y; y = &x; *y y printf("%d", *y); printf("%d", y);

При объявлении указателя ставится символ *, если несколько указателей объявляются в одной строке, то * ставится перед каждым названием.


 	int *a,b,c; //указатель только переменная a, b и c обычные переменные
 	int *a, *b, *c; //три указателя

Зачем же нужны указатели?

  • Указатель даёт нам прямой доступ к адресу переменной. Если по адресу мы изменим значение, то и в переменной оно изменится.
  • Через указатели мы можем менять значения в переменных, переданных в функцию.
  • Указатели позволяют сохранить сразу несколько результатов в одной функции.

Рассмотрим пример

						
void set (int x)
{
	x=100;
}

int main()
{
	int x=10;
	set(x);
	printf("x = %d", x);
}

В результате работы данной программы будет выведено x = 10, т.к. x в функции set и x в функции main, это две разные переменные. И изменить значение в x из main таким образом мы не можем.

Воспользуемся указателем.

						
void set (int* x) //передаем указатель в функцию
{
	*x=100; //меняем значение через указатель
}

int main()
{
	int x=10;
	set(&x); //передаем адрес
	printf("x = %d", x);
}

В данном случае значение изменилось.

Получим сразу 4 результата в одной функции.

						
void function (int a, int b, int* sum, int* raz, int* umn, double* del) //передаем в функцию 2 переменные и 4 указателя
{
	*sum = a + b;
	*raz = a - b;
	*umn = a * b;
	*del = (double)a / b;
}

int main()
{
	int a=5, b=10;
	int s, r, u, d;
	function(a, b, &s, &r, &u, &d);
	printf("sum = %d\n", s);
	printf("raz = %d\n", r);
	printf("umn = %d\n", u);
	printf("del = %d\n", d);
}

Как отправить массив в качестве аргумента?

Как вы помните, массив - это несколько переменных одного типа идущих последовательно. Зная адресс нулевого элемента массива можно обратиться к любому элементу. т.к. они расположены последовательно. При необходимости передать массив в функции используется указатель.

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
void printArray(int* arr,int size) //массив передается через указатель
{
	for(int i=0; i<size; i++)
	{
		printf("mas[%d]=%d\n",i,arr[i]); //в функции обращение к элементам массива происходит как обычно, * не нужна
	}
}

void randomArray(int* arr,int size) //массив передается через указатель, поэтому возвращать заполненный массив не нужно
{
	srand(time(NULL));
	for(int i=0; i<size; i++)
	{
		arr[i] = rand() % 50;
	}
}
int main()
{
	int mas[10];
	randomArray(mas, 10);
	printArray(mas, 10);
	return 0;
}
