Обзор условий. Понятие истинного и ложного условия

Довольно часто на поставленный вопрос человек получает ответ «да» или «нет». В зависимости от ответа он определяет свои действия и выполняет одну или другую команду (группу команд).
Роботы и другие технические устройства тоже могут выполнять различные действия в зависимости от условия. Если условие истинно (на вопрос получен ответ "Да"), то выполняются одни действия, если ложно(ответ "Нет"), то другие.

Алгоритмическая конструкция ветвление обеспечивает выполнение одной или другой последовательности команд в зависимости от истинности или ложности некоторого условия.

Рассмотрим пример: есть две переменные a=2, b=3 , сравним их:
1. a < b Действительно a меньше b, т.е. получили ответ "Да" - выражение истинно. В программировании обозначется ключевым словом true
1. a < b В даннос случае a не меньше b, т.е. получили ответ "Нет" - выражение ложно. В программировании обозначется ключевым словом false

Оператор if

Основной конструкцией ветвления в языке Си является оператор if

Блок-схема оператора if

Принцип работы:
сначала проверяется уловие, если условие истинно (true), то операторы(какой-то код) выполняются, если условие ложно(false), то операторы пропускаются.

Общий вид:

						
if(условие)
{
	операторы; //  если здесь будет одна строчка кода, то фигурные скобки {} можно не ставить.
}

Если в качестве операторов будет только одна строчка кода, то допустимо не ставить фигурные скобки{}.
Для лучшей читаемости кода необходимо ставить фигурные скобки ВСЕГДА!!!

Пример: проверка является ли число положительным

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if( x > 0)
	{
		printf("%d положительное число/n",x);
	}
	return 0;
}

Если будет введено положительное число, например 5, то получим вывод: 5 положительное число, если же будет введено отрицательное число, то ничего не выведется.
Изменим код программы так, чтобы выводился результат при любом введенном числе.

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL, "Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if( x > 0)
	{
		printf("%d положительное число/n",x);
	}
	if( x < 0)
	{
		printf("%d отрицательное число/n",x);
	}
	if( x == 0)
	{
		printf("введен ноль/n");
	}
	return 0;
}

Программа рабочая, но данное решение не самое лучшее, а без связки if-else по-другому не сделаешь!

Связка if-else

Блок-схема оператора if-else

Принцип работы:
сначала проверяется уловие, если условие истинно (true), то выполняются операторы1, если условие ложно(false), то выполняются операторы2.

Общий вид:

						
if(условие)
{
	операторы1; // выполняются, если условие истинно
} 
else
{
	операторы2; //выполняются, если условие ложно
}

Всегда, когда это возможно, несколько операторов if необходимо заменять на if-else.

Проверим положительное или отрицательное число введено при помощи связки if-else.

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL, "Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if( x > 0)
	{
		printf("%d положительное число/n",x); //выполняется, если условие истинно
	}
	else
	{
		printf("%d отрицательное число/n",x); //выполняется, если условие ложно
	}
	return 0;
}

Подумайте, что выведет программа, если ввести 0?

Изменим программу для правильной работы при любом введенном числе:

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL, "Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if( x > 0)
	{
		printf("%d положительное число/n",x); //выполняется, если первое условие истинно
	}
	else //переходим в else, если первое условие не верно
	{
			if( x < 0)
			{
				printf("%d отрицательное число/n",x); //выполняется, если второе условие истинно
			}
			else
			{
				printf("введен ноль/n"); //выполняется, если второе условие ложно
			}
	}
	return 0;
}

Может показаться, что конструкция с двумя блоками if-else сложнее, чем три независящих друг от друга блока if, но код с if-else будет работать быстрее. В лучшем случае в программе с if-else выполнится одна логическая проверка, в худшем - дву, в коде же с операторами if всегда три проверка. Любая логическая проверка затратна по времени.
При написании любой программы всгда нужно учитывать быстродействие и затраты оперативной памяти. Необходимо стремиться к повышению быстродействия (например, заменяя if на if-else) и к сокращению затрат оперативной памяти(например, не создавать лишние переменные).

Логические операторы

При записи логическим выпожений всегда используется один из логических операторов.


Логические операторы

Знак Название Описание Пример
> больше возвращает true, если число слева, больше чем справа, иначе - false a>b
< меньше возвращает true, если число слева, меньше чем справа, иначе - false a<b
>= больше либо равно возвращает true, если число слева, больше либо равно чем справа, иначе - false a>=b
<= меньше либо равно возвращает true, если число слева, меньше либо равно чем справа, иначе - false a<=b
== проверка на равенство возвращает true, если числа равны, иначе - false a==b
!= проверка на не равенство возвращает true, если числа не равны, иначе - false a!=b
