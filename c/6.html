Группировка условий через логическое И и ИЛИ.

Довольно часто возникает необходимость проверить сразу несколько условий. Например, нам нужно чтобы число было больше 5, но меньше 20. Решим эту задачу уже имеющимися знаниями:

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if(x > 5)
	{
		if(x < 20)
		{
			printf("%d больше 5 и меньше 20/n",x);
		}
	}
	return 0;
}

У нас два условия и пришлось написать два блока if. А ведь условий может быть и 3 и более. Чтобы не писать такие сложные конструкции есть ещё два логических оператора, о которых вы пока не знаете. Это оператор &&(логическое И) и оператор ||(логическое ИЛИ).

В таблице показан принцип работы этих операторов

true
true		 true
false		 false
true		 false
false
&& true false false false
|| true true true false

Как видно из таблицы оператор && дает нам истинно, если оба выражения истинны, оператор || дает нам истинно, если истинно хотя бы одно из условий.

У оператора && приоритет выше, чем у ||. Приоритет выше - значит оператор выполняется раньше. Это как в математике: приоритет у знаков * и / выше чем у + и -.

Изменим нашу задачу, используя полученные знания: 

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	int x;
	printf("введите число");
	scanf("%d",&x);
	if(x > 5 && x < 20)
	{
			printf("%d больше 5 и меньше 20/n",x);
	}
	return 0;
}

Используя операторы && и || можно записывать сложные условия, состоящие из нескольких выражений в один оператор if.

  • Напишите программу, которая запрашивает у пользователя номер месяца и затем выводит соответствующее название времени года.
    В случае, если пользователь введет недопустимое число, программа должна вывести сообщение "Ошибка ввода данных".

#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL, "Russian");
	int n;
	printf("Введите номер месяца: ");
	scanf("%d", &n);
	if (n >= 2 && n <= 4)
	{
		printf("весна\n");
	}
	else
	{
		if (n >= 5 && n <= 7)
		{
			printf("лето\n");
		}
		else 
		{
			if (n >= 8 && n <= 10)
			{
				printf("осень\n");

				{
					printf("зима\n");
				}

					printf("Ошибка ввода данных\n");
				}
			}
		}

  • Дракон отращивает по 3 головы каждый год, но после того, как ему исполнится 100 лет - только по 2. сколько голов будет у дракона через n лет

    • 						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL, "Russian");
	int n;
	printf("Введите количество лет: ");
	scanf("%d", &n);
	int heads = 0;
	if (n <= 100)
	{
		heads = n * 3;
	}
	else 
	{
		heads = 100 * 3;
		heads += (n - 100) * 2
	} 
	printf("heads count = %d\n", heads);
	return 0;
}
