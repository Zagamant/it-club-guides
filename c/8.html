Понятия цикла с предусловием и постусловием.

В жизни мы часто сталкиваемся с действиями которые повторяются много раз. Например, каждый вечер мы ложимся спать или каждое утро мы просыпаемся и завтракаем. В программировании также возникает необходимость повторять каки-то действоия много раз. Для решения этой задачи были придуманы циклы.

Цикл - это часть программы, которая повторяется необходимоое количество раз.

Существует две разновидности циклов: с предусловием и постусловием.
В цикле с предусловием, сначала проверяется условие и если оно верно, то выполняются действия. В цикле с постусловием, сначала выполняются действия, а затем проверяется условие.
Главное отличие этих циклов: цикл с постусловием выполнится хотя бы один раз, цикл с предусловием может ни разу не выполниться.

Циклы while и do-while.

В таблице показан принцип работы этих операторов

название while(пока, оператор с предусловием) do-while(делать-пока, оператор с постусловием)
блок-схема
общий вид 
						
	while(условие)
	{
		операторы;
	}
						
		
						
	do
	{
		операторы;
	}while(условие);
принцип работы сначала проверяется условие, если условие верно, то выполняются операторы и снова проверяется условие и т.д. пока условие не нарушится сначала выполняются операторы, затем проверяется условие, если условие верно, то снова выполняются операторы и т.д. пока условие не нарушится
пример:
вывод чисел от 1 до 10		 
						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	int x=1;
	while(x <= 10)
	{
		printf("%d ",x);
		x++;
	}
	return 0;
}
						
		
						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	int x=1;
	do
	{
		printf("%d ",x);
		x++;
	}while(x <= 10);
	return 0;
}

Как видно из таблицы цикл while от цикла do-while отличается только расположением условия. Цикл do-while выполнится хотя бы один раз, цикл do-while может ни разу не выполнится, если изначально условие окажется не верныым. Циклы взаимозаменяемы, т.е. всегда один цикл можно заменить на другой, но в разных ситуациях удобно использовать разные циклы.

Бесконечные циклы - проблемы и способы использования.

Бесконечный цикл — цикл, написанный таким образом, что условие выхода из него никогда не выполняется. О программе, вошедшей в бесконечный цикл, иногда говорят, что она зациклилась

Самый простой вариант создания бесконечного цикла написать в условии true.
Программа бесконечно выводит "привет".

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	while(true)
	{
		printf("привет");
	}
	return 0;
}

Эта же задача выполнена с помощью цикла do-while

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	do
	{
		printf("привет");
	}while(true)
	return 0;
}

Бесконечный цикл опасен - он может привести к зависанию компьютера. Если используется конструкция с бесконечным циклом всегда нужно предусмотреть какое-то условие для выхода из этого цикла.

Оператор break используется для принудительной остановки цикла. (Можно использовать для любого цикла)

Программа считает сумму всех вводимых чисел пока не будет введен 0.

						
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
int main()
{
	setlocale(LC_ALL,"Russian");
	int n, sum=0;
	
	while(true)
	{	
		printf("введите чило: ");
		scanf("%d", &n);
		if(n == 0)
		{
			break; //принудительная остановка цикла
		}
		sum+=n;
	}
	printf("сумма всех введенных чисел = %d", sum);
	return 0;
}

#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
#include <stdlib.h> // библиотека стандартных функций
#include <time.h> //библиотека работы со временем
int main()
{
	srand(time(NULL));//мы даем генератору случайных чисел начальные значения в виде текущего времени
	int i=1;
	while(i<10)
	{
		int n = rand();
		printf("%d\n",n);
		i++;
	}
	//чтобы генерировать числа в неком диапазоне, запомнить!
	printf("------------------------------\n");
	//генерация от 0 до max - rand()%max;
	//пример - генерация от 0 до 131
	i=1;
	while(i<10)
	{
		int n = rand()%131;
		printf("%d\n",n);
		i++;
	}
	printf("------------------------------\n");
	//генерация от min до max 
	// min + rand() % (max-min);
	//пример - генерация от 5 до 15
	i=1;
	while(i<10)
	{
		int n = 5 + rand()%(15-5);
		printf("%d\n",n);
		i++;
	}
	// min + rand() % (max-min);
	//пример - генерация от min до max c клавиатуры
	i=1;
	int min, max;
	scanf("%d%d",&min,&max);
	while(i<10)
	{
		int n = min + rand()%(max-min);
		printf("%d\n",n);
		i++;
	}
	return 0;
}
